🚀 QUICK START (5 MINUTES)
═══════════════════════════════════════════════════════════════════════════════

1. Create Project:
   npm create vite@latest StudyFlowAI -- --template react

2. Install Dependencies:
   npm install tailwindcss postcss autoprefixer framer-motion recharts \
   lucide-react react-hot-toast firebase axios clsx react-router-dom

3. Setup Tailwind:
   npx tailwindcss init -p

4. Copy Source Code:
   Copy all files from STUDYFLOW-COMPLETE-CODE-PART1-4.txt into src/

5. Create .env:
   VITE_FIREBASE_API_KEY=xxx
   VITE_FIREBASE_AUTH_DOMAIN=xxx
   VITE_FIREBASE_PROJECT_ID=xxx
   VITE_FIREBASE_STORAGE_BUCKET=xxx
   VITE_FIREBASE_MESSAGING_SENDER_ID=xxx
   VITE_FIREBASE_APP_ID=xxx
   VITE_GEMINI_API_KEY=xxx

6. Start Server:
   npm run dev

7. Open Browser:
   http://localhost:3000
