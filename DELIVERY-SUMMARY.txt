🚀 5-MINUTE QUICK START
═══════════════════════════════════════════════════════════════════════════════

Step 1: Create Project
────────────────────
npm create vite@latest StudyFlowAI -- --template react
cd StudyFlowAI

Step 2: Install Dependencies
─────────────────────────────
npm install
npm install tailwindcss postcss autoprefixer framer-motion recharts \
  lucide-react react-hot-toast firebase axios clsx react-router-dom

Step 3: Setup Tailwind
──────────────────────
npx tailwindcss init -p

Step 4: Copy Source Code
────────────────────────
• Copy all code from STUDYFLOW-COMPLETE-CODE-PART1.txt
• Copy all code from STUDYFLOW-COMPLETE-CODE-PART2.txt
• Copy all code from STUDYFLOW-COMPLETE-CODE-PART3.txt
• Copy all code from STUDYFLOW-COMPLETE-CODE-PART4.txt
Into their respective src/ subdirectories

Step 5: Create .env File
─────────────────────────
VITE_FIREBASE_API_KEY=your_key
VITE_FIREBASE_AUTH_DOMAIN=your_domain
VITE_FIREBASE_PROJECT_ID=your_project
VITE_FIREBASE_STORAGE_BUCKET=your_bucket
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_FIREBASE_APP_ID=your_app_id
VITE_GEMINI_API_KEY=your_gemini_key

Step 6: Run Dev Server
──────────────────────
npm run dev

Step 7: Open Browser
─────────────────────
http://localhost:3000

Done! 🎉
