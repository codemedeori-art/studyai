## 🎯 GETTING STARTED (5 MINUTES)

```bash
# 1. Create project
npm create vite@latest StudyFlowAI -- --template react

# 2. Navigate to project
cd StudyFlowAI

# 3. Install all dependencies
npm install tailwindcss postcss autoprefixer framer-motion \
  recharts lucide-react react-hot-toast firebase axios \
  clsx react-router-dom

# 4. Initialize Tailwind
npx tailwindcss init -p

# 5. Copy all source files into src/ directory

# 6. Create .env file with credentials

# 7. Start development server
npm run dev

# 8. Open http://localhost:3000 in browser
```

## 🔐 ENVIRONMENT SETUP

Create `.env` file in project root:

```env
VITE_FIREBASE_API_KEY=your_key
VITE_FIREBASE_AUTH_DOMAIN=your_domain
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_bucket
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_FIREBASE_APP_ID=your_app_id
VITE_GEMINI_API_KEY=your_gemini_key
```

Get these from:
- Firebase Console: https://console.firebase.google.com
- Gemini API: https://aistudio.google.com/app/apikey

## 🎓 CORE CONCEPTS

### Authentication Flow
1. User signs up with email or Google
2. Firebase creates user account
3. Session stored in browser
4. Protected routes check auth
5. User can logout

### Data Flow
1. User inputs data (e.g., study plan)
2. Data sent to Firestore
3. Real-time listeners update UI
4. User sees updated data

### AI Integration
1. User sends prompt to AI Assistant
2. Gemini API processes request
3. AI returns response
4. Chat saved to Firestore

### Component Hierarchy
```
App
├── AuthProvider
│   └── ThemeProvider
```
