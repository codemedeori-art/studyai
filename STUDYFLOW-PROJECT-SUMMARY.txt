## 🚀 QUICK START

### In 5 Minutes:

```bash
1. cd d:\ai
2. npm create vite@latest StudyFlowAI -- --template react
3. cd StudyFlowAI
4. npm install tailwindcss postcss autoprefixer framer-motion recharts lucide-react react-hot-toast firebase axios clsx react-router-dom
5. npx tailwindcss init -p
6. Copy all source code files
7. Create .env with Firebase/Gemini keys
8. npm run dev
```

That's it! Your app is running.

## 💾 DATABASE SCHEMA

### Users Collection
```javascript
{
  id: user.uid,
  email: string,
  displayName: string,
  createdAt: timestamp,
  preferences: {...}
}
```

### Tasks Collection
```javascript
{
  id: string,
  userId: string,
  title: string,
  completed: boolean,
  dueDate: date,
  createdAt: timestamp
}
```

### Habits Collection
```javascript
{
  id: string,
  userId: string,
  name: string,
  frequency: "daily|weekly|monthly",
  streak: number,
  completed: [dates],
  createdAt: timestamp
}
```

### Study Plans Collection
```javascript
{
  id: string,
  userId: string,
  subjects: [strings],
  examDates: [strings],
  weakTopics: [strings],
  availableHours: number,
  schedule: string (AI generated),
  createdAt: timestamp
}
```

### Chats Collection
```javascript
{
  id: string,
  userId: string,
  message: string,
  response: string,
  timestamp: timestamp
}
```

### Analytics Collection
```javascript
{
  id: string,
  userId: string,
  totalHours: number,
  productivity: number,
  streakDays: number,
  subjectsStudied: number,
  updatedAt: timestamp
}
```
